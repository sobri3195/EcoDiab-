# EcoDiab

EcoDiab is a production-ready demo SPA for **AI-Powered Green Diabetes Management** built with React + TypeScript + Tailwind.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

- Import this repo in Vercel.
- Framework preset: **Vite**.
- `vercel.json` includes SPA rewrite to `index.html`.
